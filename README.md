# Redbull-Women-Audience-Analysis
Data analysis of  Red Bull India's women audience gap - 500 respondents, excel dasboard, insight & recommendations

## Project Overview
Analyzed 500 Indian women's perception,awareness and purchase behavior towards Red Bull India.

## Key Findings
- 78 women cited Health Concerns as #1 barrier
- 67 women feel Red Bull is'Not for Women'
- Kolkata has lowest brand awareness

## Tools Used
- Microsoft Excel
- Pivot Tables & Charts
- Data Cleaning

## Dataset
500 Indian women respondents across 
10 cities
